面向对象程序设计的基本特点：抽象 封装 继承 多态
类：面向对象编程里，程序模块由类构成，类是对逻辑关联的函数和数据的封装，是问题的抽象描述
类的定义：
class 类的名称
{
	public：公有成员	
	private：私有成员
	(protected)：保护成员
};
对象是类的具体的实现
创建对象：类名  对象名
对象调用成员函数：对象名.成员函数名
成员函数的实现：返回值类型 类名::函数名(形参表)

构造函数：在对象被创建时自动调用，为对象初始化为一个特定状态
无需提供参数的构造函数称为默认构造函数，如果类中没有构造函数系统会自动生成默认构造函数
委托构造函数：将自身职责委托给同类的其他构造函数
复制构造函数：形参是本类对象的引用，用一个已经存在的对象去初始化同类的一个新对象
如果类中没有复制构造函数系统会自动生成隐含复制构造函数
复制构造函数的调用场景：
1.用类的一个对象去初始化该类的另一个对象
2.函数的形参是类的对象，调用函数时进行形参和实参的结合
3.函数的返回值是类的对象

浅复制深复制和移动构造
1. 核心概念对比
假设我们有一个类(class)，里面包含一个指针，指向堆内存（Heap）中的数据
概念	浅复制 (Shallow Copy)	        深复制 (Deep Copy)	                       移动构造 (Move Constructor)
行为	只复制指针地址	申请新内存       复制数据内容	                               窃取指针，原指针置空
结果	两个对象指向同一块内存	        两个对象拥有独立且相同的内存	       新对象拥有内存，旧对象变为空壳
速度	极快	                                        较慢 (涉及内存分配和拷贝)	       极快
危险 高 (容易造成Double Free)	        低 (安全)	                                       低 (但旧对象不能再用了)
							重新分配新内存				       原指针要置空避免野指针
左值和右值：
左值：有变量名持久存在(能够寻址的)          左值引用：只能绑定左值 T&
右值：没有变量名临时存在(不能寻址的)       右值引用：只能绑定右值 T&&
一般使用常量左值绑定：const T&  能绑定左值右值和常量左值
析构函数：对象生存期结束时被自动调用
~ 类名( ){函数体}
default和delete函数
default：=default让系统自动生成默认合成的构造函数，复制构造函数和析构函数
delete：=delete除了析构函数外都可使用，删除对应的函数以禁止类的操作

类的静态成员
静态数据成员：类属性是类的所有对象共有特征，对任何对象而言的属性值是相同的
在类的定义中对静态数据成员进行声明，在类外进行定义和初始化
private：static 数据类型 变量名}
数据类型 类名::变量名=值
本质原因：需要在文件作用域的地方为静态生存期的变量单独分配内存空间
常静态数据成员：直接在类定义里进行定义和初始化
const static 数据类型 变量名=值
静态函数成员：可以通过类名而非对象直接调用成员函数，实现了同类不同对象共享
访问非静态成员必须通过对象调用，因为这是不同对象特有的属性值
{static 数据类型 函数名( ){函数体}}
类名::函数名( )

类的友元
友元函数：可以在类外通过对象名访问类的私有和保护成员
friend 返回值类型 函数名(形参表)
友元类：该类的所有成员函数都是另一个类的友元函数
class A
{public：friend class B}//声明B是A的友元类
在B的成员函数中可以直接通过象名访问A的私有成员
友元关系单向且不能传递

const修饰类的成员
常数据成员
class A
{private:
const int a;//常量数据成员
static const int b}//静态常数据成员
const int A::b=10//静态常数据成员在类外初始化
A::A(int i): a(i)//常量数据成员
常成员函数
返回值类型 函数名(参数表) const
1.常对象只能通过常函数调用
2.调用常函数的对象一律视为常对象
3.const可以实现对函数的重载

1.对象指针
语法：类名 *对象指针名=&对象名
通过对象指针访问成员：对象指针名->成员名
2.this指针：this用于指向正在被成员函数操作的对象
(this指针实际上是每个成员函数的一个隐含参数，调用成员函数的时候，目的对象的地址会自动作为该隐参)
3.指向类的非静态成员的指针
声明：数据类型 类名::*指针名                       //指向数据成员
	  数据类型 (类名::*指针名)(参数表)        //指向函数成员
定义：指针名=&类名::成员名 
实现：对象名.*成员指针名//对象指针名->*成员指针名                                        //数据成员
	  (对象名.*成员指针名)(参数表)//(对象指针名->*成员指针名)(参数表)           //函数成员
4.指向类的非静态成员的指针
类的非静态成员不依赖于对象，直接使用指针和函数指针访问即可
